# Website

Dieser Ordner enthält eine React- und TypeScript-Oberfläche, die ein hochgeladenes echtes Bild mit mathematischen Ansätzen beschreibt.

## Funktionen
- Bild-Upload
- Methodenauswahl: Polynom, Gauß/RBF, Fourier
- Saubere mathematische Formeldarstellung mit LaTeX
- Anzeige des 4×4-Abtastgitters und der Farbproben

## Befehle
```bash
npm install
npm run dev
npm run build
```
